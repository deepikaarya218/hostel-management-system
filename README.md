# Hostel Complaint Management System

A full-stack web application for managing hostel complaints efficiently.

---

## 🚀 Overview
This project allows students to raise and track hostel-related complaints such as plumbing, electricity, cleanliness, and other issues. Admins can manage and update the status of each complaint.

---

## ✨ Features
- Student Registration & Login
- Raise Complaints with category and priority
- View previous complaints
- Track complaint status (Pending / In Progress / Resolved)
- Simple and user-friendly interface

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB

---

## ⚙️ How It Works
1. Student submits a complaint through the form  
2. Backend API saves the data in MongoDB  
3. Complaints are fetched and displayed on the frontend  
4. Admin can manage and update complaint status
